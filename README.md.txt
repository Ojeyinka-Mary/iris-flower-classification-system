# Iris Flower Classification System

## Project Description
This project is an AI-powered machine learning application that predicts the species of an Iris flower based on flower measurements.

## Problem Being Solved
Identifying flower species manually can be time-consuming. This system automates the classification process using machine learning.

## Technologies Used
- Python
- Scikit-Learn
- Joblib
- Streamlit

## How the System Works
The user enters:
- Sepal Length
- Sepal Width
- Petal Length
- Petal Width

The trained Random Forest model predicts the flower species.

## Model
Random Forest Classifier

## Deployment
Built using Streamlit.

## Running Locally

```bash
pip install -r requirements.txt
streamlit run app.py
